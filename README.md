 📧 Email Spam Detection System

🚀 Project Overview
This project is a Machine Learning-based Email Spam Detection System that classifies messages as Spam or Not Spam (Ham) using Natural Language Processing (NLP).

🎯 Key Features
- Detects spam messages with good accuracy  
- Uses NLP for text processing  
- Fast and simple prediction system  

🛠️ Technologies Used
- Python  
- Machine Learning  
- NLP  
- Scikit-learn  
- Pandas  

📂 Project Structure
spam_detector.py - Main code  
spam_model.pkl - Trained model  
vectorizer.pkl - Text vectorizer  
spam.csv - Dataset  

⚙️ How It Works
1. User enters message  
2. Text is processed  
3. Model predicts  
4. Output: Spam or Not Spam  

 ▶️ How to Run
pip install pandas scikit-learn  
python spam_detector.py  

 📊 Sample Output
Input: You won a prize  
Output: Spam  

💡 Use Cases
- Email filtering  
- SMS spam detection
